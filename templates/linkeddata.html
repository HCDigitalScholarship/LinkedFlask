HOME

This is Maddy Hodges first user interface for Haverford's Quaker Linked Data Project.

Currently the project only has the data from the Cope Evans Project and there are plans to incorporate three other Quaker projects. Linked data is data that is structured in a way that it can be interlinked with other data through semantic queries. Luckily you wont have to learn how to write semantic queries to utilize the data. I have created a interface with which you can explore the data. The searches I have available at the moment are in the tabs above.

Maybe I'll use the sticky footer with a fixed navbar if need be.
